## Database File Folder
This folder preserves all corresponding data for creating World Government Bonds Market Yield Database Project that represent
both fact and dimension tables.  
There are two different fact tables within this folder, namely:  
- fact_globalbonds_rate : Fact table that presents government bonds yield data in percentages point rate;
- fact_globalbonds_bps  : Fact table that presents government bonds yield data in basis points (bps). In most Economic and Finance
  environment, 100 basis points is equivalent to 1.0 percentages point.
